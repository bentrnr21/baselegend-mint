# 🧙‍♂️ BaseLegend DAO

BaseLegendDAO is the governance layer of the BaseLegend project — a strategic, on-chain card game powered by Base.

Holders of DAO tokens will be able to:
- Propose improvements to the game
- Vote on new card releases and game mechanics
- Manage treasury and collaborations
- Shape the future of the BaseLegend universe

Built using [Nouns Builder](https://nouns.build) on Base.
